# Fufaji Da Pizza

This is a basic pizza ordering website

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the requirements. You can also use pipenv

## Usage

```
$env:FLASK_APP = "app.py"
flask run
```
